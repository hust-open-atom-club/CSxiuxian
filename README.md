# CSxiuxian

计算机修仙境界指南 —— 通过修仙比喻学习计算机科学

## 项目简介

CSxiuxian 是一个创意项目，将计算机科学的学习过程比喻为修仙修炼。通过层层境界突破，帮助学习者系统化掌握编程、算法、系统设计等技能。站点使用 MkDocs + Material 主题构建，提供美观、易导航的文档体验。

## 主要功能

- **境界体系**：从锻体期到天道圣人，系统划分计算机技能等级
- **修炼指南**：详细突破指南和修炼心法
- **神话传说**：探讨AI、工程等话题的修仙视角
- **现代化界面**：Material Design 风格，支持暗色模式

## 部署指南

### 环境要求

- Python 3.8+
- pip

### 安装步骤

1. **克隆仓库**
   ```bash
   cd CSxiuxian
   ```

2. **安装依赖**
   ```bash
   pip install mkdocs mkdocs-material mkdocs-minify-plugin mkdocs-material-extensions
   ```

3. **本地预览**
   ```bash
   mkdocs serve
   ```
   打开浏览器访问 `http://127.0.0.1:8000` 查看站点
