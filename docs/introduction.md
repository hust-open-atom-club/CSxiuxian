# 引言：为什么会有修仙境界？

我作为野路子（披毛戴角，湿生卵化）出身的程序员，一直希望弄明白我和那些天才少年的差距究竟在哪里。直到我踩了各种各样的坑，经历了各种各样的事，加上本人对于修仙小说的喜爱，总结出了该计算机修仙境界。

## 境界划分

因此，按照修仙小说常见的境界设定，我们把修真境界分成以下几种：

- **锻体期**（凡间武者）
- **炼气期**
- **筑基期**
- **金丹期**（金丹一出仙凡别）
- **元婴期**
- **化神期**
- **至尊**（图灵榜证道，天道境）（或者洪荒小说中的圣人）

同样，也有仙人的设定。仙人是获得果位的修真者，实力大概在金丹期到化神期不等。

## 重要说明

!!! note "境界划分注记"
    - 这里的境界划分相当宽泛，炼气巅峰实际上也算强者了。我的主要目的是展示计算机不同等级强者之间的边界。
    - 炼气巅峰不一定比筑基弱，筑基大圆满若机缘了得也可击败金丹。
    - 修仙是相当残酷的。这也是用修仙来进行比喻的原因。
    - 修仙境界是以实力评价。即使一个北大本博取得博士学位，但是如果干的还是炼气期的活，找的是炼气期的工作，那么就算作炼气期。
    - **并非严肃讨论，仅供图一乐。**
