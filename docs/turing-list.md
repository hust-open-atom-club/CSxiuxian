# 图灵榜

!!! quote "天道圣人"
    图灵榜归属于天道，一旦在图灵榜上留下真名，就可万劫不灭了。这也是整个修仙界的最终目标。

## 图灵榜简介

图灵榜是计算机科学领域的最高荣誉，由 ACM（Association for Computing Machinery）颁发的图灵奖。该奖项被誉为“计算机界的诺贝尔奖”，旨在表彰那些对计算机科学领域做出杰出贡献的个人。

## 获奖条件

- 对计算机科学理论和技术的发展产生深远影响
- 创新性贡献
- 长期影响

## 著名获奖者

- **Alan Turing** (1974) - 奠基了计算机科学的基础
- **Donald Knuth** (1974) - 《计算机程序设计艺术》
- **Tim Berners-Lee** (2016) - 万维网发明者
- **Yao Qizhi** (姚期智) (2000) - 密码学和计算理论

!!! tip "国内获奖者"
    目前国内只有姚期智教授一位图灵奖得主。
